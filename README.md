 <h3 align="center">Vite React PWA Boilerplate (TypeScript)</h3>

## What's included

-   Vite
-   React
-   PWA Plugins
-   MUI
-   MUI Icons
-   Robot Font (English Language)
-   Tajawal (Arabic Language)
-   Axios
-   i18next (for translations)
-   React Router Dom

## Installation

-   Clone project
-   Run `npm install`
